compute_danger_factor_dict()[source]

Calcule le facteur de danger associ chaque espce en fonction des caractristiques comportementales du robot.

  • 0 : robot inoffensif

  • 1 : robot capable de se dfendre

  • 2 : joueur humain au comportement inconnu encore (suppos capable de se dfendre)

  • 3 : robot pouvant attaquer par opportunit

  • 4 : robot attaquant coup sr

Rq : la gestion par dict permet de diffrencier les facteurs de dangers en fonction des espces (spcisme). En V1 on standardise.
