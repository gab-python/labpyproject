Text

class labpyproject.core.pygame.widgets.Text(fontobj, text='', fgcolor='0x000000', linespacing=0, textalign='left', **kwargs)[source]

Bases : labpyproject.core.pygame.core.CustomSprite

Widget texte basique.

Constructeur

Paramtres

  • fontobj un objet pygame.font.Font (de prfrence) ou pygame.freetype.Font (support de caractres plus large mais rendu et wrap moins bons)

  • text le texte afficher

  • fgcolor couleur du texte

  • bgcolor couleur du fond (transparent par dfaut)

  • size taille du texte en points (par dfaut celle de lobjet fontobj)

  • linespacing espacement interlignes

  • textalign alignement horizontal du texte (gauche par dfaut), uniquement pour un objet pygame.font.Font

Rq : clipping implmenter dans draw_display, en subclassant CustomSprite.get_display_dest pour prendre en compte les diffrences de rects de publication.
