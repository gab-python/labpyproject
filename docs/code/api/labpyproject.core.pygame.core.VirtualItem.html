VirtualItem

class labpyproject.core.pygame.core.VirtualItem(**kwargs)[source]

Bases : labpyproject.core.pygame.core.BoxModelObject

BoxModelObject supportant les attributs de gestion des z-indexs (couches) ainsi que les proprits et mthodes de base de la logique de publication.

Virtual indique que cet objet na pas de surface propre (au sens pygame) et quil nest pas un objet de publication rel.

Constructeur :

Paramtres

**kwargs peut contenir local_layer ainsi que toutes les proprits de boxing supportes par BoxModelObject.
