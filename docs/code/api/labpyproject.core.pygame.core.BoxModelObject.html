BoxModelObject

class labpyproject.core.pygame.core.BoxModelObject(**kwargs)[source]

Bases : object

Super classe portant les proprits et mthodes fondamentales du box model.

  • position au sens css du terme soit fixed, absolute, static, relative

  • attributs locaux pouvant tre exprims en pixel ou pourcentage : x, y, top, left, bottom, right, width, height, padding, margin

  • bornage des dimensions : minwidth, maxwidth, minheight, maxheight

  • attributs dalignement : align, valign

  • attributs dadaptation au contenu : snapW, snapH

  • flex (int, default None) : valeur de flexibilit si le conteneur parent impose un layout (horizontal ou vertical lheure actuelle)

Remarques:

  • lorigine du repre se situe dans le coin haut/gauche, laxe des absisses est oriente vers la droite, celui des ordonnes vers le bas

  • les metrics en % sont calculs par rapport la surface aloue par le parent

  • la diffrence de css, width et height reprsentent les dimensions totales de la box, padding et margin inclus. Les dimensions du contenu et du fond (ou de la bordure), pouvant tre moindres.

  • padding et margin sont uniformes sur chacun des bords (V1 : allge les calculs)

  • position fixed est gre partiellement par les RealContainers du fait du clipping de leur surface de publication.
