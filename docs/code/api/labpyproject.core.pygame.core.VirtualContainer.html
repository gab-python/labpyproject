VirtualContainer

class labpyproject.core.pygame.core.VirtualContainer(bgcolor=None, **kwargs)[source]

Bases : labpyproject.core.pygame.core.VirtualItem

VirtualItem dt des capacits dun conteneur :

  • publication : ajout / suppression dobjets enfants

  • propagation du resize aux objets enfants

  • layouts basiques : via la proprit direction :

    • direction = None : layout absolu (canvas), cas par dfaut

    • direction = horizontal : layout des lments de position fixed ou absolute selon laxe x

    • direction = vertical : respectivement selon laxe y

Virtual indique que cet objet na pas de surface propre (au sens pygame). Comme VirtualItem il ne sagit pas dun objet de publication rel. Par contre il sinscrit dans la gestion gnrique des couches et de la publication. Sa descendance sera publi rellement par un RealContainer parent (Root par dfaut).

faire

Support de la visibilit, actuellement effectif uniquement pour les composants disposant dune surface de publication.

Constructeur

Paramtres

  • bgcolor (str) couleur hxa

  • **kwargs peut contenir direction ainsi que toutes les proprits de VirtualItem.
