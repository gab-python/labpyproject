xlook_for_cases_impacted(casedanger, count=0, dictimpact=None)[source]

Identification rcursive des cases impactes par lactivation dun danger

Retourne un dictionnaire :

  • dictimpact[danger_done] : usage interne

  • dictimp[flat_list] : liste (set) complte des cases impactes

  • dictimpact[0] : liste (set) de tupples (case, impact) des cases touches au pas 0

  • dictimpact[n] : liste (set) de tupples (case, impact) des cases touches au pas n

avec n : nombre de pas danimation.

Rq importante : les donnes mises en cache ne comprennent pas le danger initial. En effet lors des calculs de commandes automatiques, on simule des explosions avec des grenades virtuelles. On ninclue le danger rel quau dernier moment.
