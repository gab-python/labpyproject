    Accueil de la documentation

    labpyproject c'est:

    • Un prototype de jeu d'arcade dans lequel il faut s'échapper d'un labyrinthe, peuplé de bots plus ou moins bien intentionnés, rempli de mines et de bonus. Cliquez sur JEU pour en savoir plus sur son installation et son fonctionnement.
    • Des développements python 3, mais aussi des recettes techniques pour optimiser Numpy avec OpenBLAS, distribuer des versions exécutables d'une application python. Cliquez sur PROJET pour en savoir plus sur l'origine du projet, accéder à la documentation du code source et consulter les recettes techniques.
